# Summer school online day2
CSS types I used:
- Inline CSS: Added directly to the <body> tag for background color.
- Internal CSS: Defined in the <style> section inside the <head>.
- External CSS: Defined in style.css and linked to index.html.

Selectors I used:
- Element selectors: p, h1, h2, ul
- ID selector: #profile-pic
- Class selector: .profile-card, .bio, .contact
- Group selectors: h1, h2
- Descendant selector: .contact p
- Attribute selector: .contact a[href]

Bonus:
- Added a hover effect for contact links using a[href]:hover.
